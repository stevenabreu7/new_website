# Personal Website

This is a personal website / blog using [Gatsby](https://www.gatsbyjs.org/), [Markdown](https://www.markdownguide.org/), [Typescript](https://www.typescriptlang.org/), [Styled Components](https://styled-components.com/), [Talwind CSS](https://tailwindcss.com/), [Framer Motion](https://www.framer.com/motion/) & [React Font Awesome](https://github.com/FortAwesome/react-fontawesome/). Forked from [this repo](https://github.com/SaimirKapaj/gatsby-markdown-typescript-personal-website).

## Quick start

- clone this repository
- start development environment at `http://localhost:8000`
```shell
gatsby develop
```
- get an optimized production build
```shell
gatsby build
```
- local server for testing production site at `http://localhost:9000`
```shell
gatsby serve
```
